New website, new danger!

Joseph Rutland

I have a new website. Same address, but… different.

The reason is that renewal with one.com was coming up and would have cost around 100GBP for the year.

To avoid this, I’ve brought it in-house, and the site is now running (along with a bunch of other stuff) from a Raspberry Pi in my living room. So I’m only paying for domain registration now.

Bear in mind that you can buy a spanking new Pi for around 30 quid, and you can see that this is quite the bargain. Even if I was only running this site on it, the Pi would have paid for itself in around three months.

Good eh?

I’ve also ditched WordPress for Jekyll*. It’s a static site builder which uses latex and markdown.

i do the editing in Atom, then push it to the Pi using scp.

Look forward to seeing much more from me here.

Email is handled using a catch-all, so if you do email me, any reply will likely come from a different address. Mail servers are hard.
